## Форма 1: Форма входа в систему для сотрудников
![Illustration](../misc/images/2.png)

### Основные операции
| Операция | Роли | Условия выполнения | Результат в текущем окне | Результат в системе |
|----------|------|-------------------|--------------------------|---------------------|
| Авторизация | Менеджер, Мастер, Админ | Корректные логин/пароль | Переход на главную страницу | Доступ к функционалу по роли |
| Валидация данных | Все | Заполнение полей | Сообщения об ошибках | - |

### Обеспечивающие операции
- **Проверка формата:** Логин (текст, 20 символов), Пароль (текст, скрытый ввод)
- **Сообщения:** "Неверный логин или пароль" при ошибке авторизации

### Данные
**Объект данных:** Пользователь системы
**Форма карточки данных:**
| Поле | Тип | Длина | Обязательность | Значение по умолчанию |
|------|-----|-------|----------------|----------------------|
| Логин | Текст | 20 | Да | - |
| Пароль | Текст | - | Да | - |

### Логирование
- **Операции:** Попытка входа (успешная/неуспешная)
- **Атрибуты:** Логин, IP-адрес, результат попытки
- **Служебные поля:** Временная метка, user agent

## Форма 3: Экран расписания мастеров
![Illustration](../misc/images/6.png)

### Основные операции
| Операция | Роли | Условия | Результат в окне | Результат в системе |
|----------|------|---------|------------------|---------------------|
| Просмотр расписания | Менеджер, Админ | Наличие мастеров | Отображение сетки | - |
| Редактирование слота | Менеджер, Админ | Слот свободен | Открытие карточки | Изменение в БД |
| Фильтрация | Менеджер, Админ | - | Обновление списка | - |

### Обеспечивающие операции
**Фильтрация:**
- Поля: Дата, Мастер, Статус
- Сохранение фильтров: Сохранять последние настройки
- Сообщение: "Нет записей по выбранным критериям"

**Поиск:**
- Поля: Имя мастера
- Точность: Частичное совпадение
- Сообщение: "Мастер не найден"

**Сортировка:**
- Поля: Дата (по возрастанию), Время, Мастер
- По умолчанию: Дата (возрастание)

### Данные
**Объект данных:** Расписание мастеров
**Форма списка данных:**
| Поле | Тип | Обязательность | Описание |
|------|-----|----------------|----------|
| Дата | Дата | Да | День расписания |
| Время | Время | Да | Временной слот |
| Мастер | Справочник | Да | Выбор из списка |
| Статус | Перечисление | Да | Свободен/Занят |

**Форма карточки данных:**
| Поле | Тип | Обязательность | Описание |
|------|-----|----------------|----------|
| Время | Диапазон | Да | 13:00-14:00 |
| Мастер | Выпадающий список | Да | Выбор мастера |
| Статус | Перечисление | Нет | Статус слота |

### Логирование
- **Операции:** Изменение расписания, бронирование слота
- **Атрибуты:** ID слота, старое/новое значение, пользователь
- **Служебные поля:** Время изменения, IP-адрес
